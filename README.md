# Lyxelandflamingo Creative Page

## Features
- Semantic HTML5
- BEM naming convention
- Fully responsive (1440px / 375px)
- Custom CSS (no frameworks)
- Google Fonts (Bebas Neue, Nunito Sans)
- Sticky navigation
- Mobile hamburger menu
- Hover animations

## Assumptions
- Exact Figma spacing approximated due to no direct layer access
- Placeholder images used (replace from /assets)

## Fonts Used
- Bebas Neue (Headings)
- Nunito Sans (Body)

## Colors
- Purple: #4d319a
- Yellow: #ffba00
